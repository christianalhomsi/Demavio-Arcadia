# ميزة الفلترة حسب نوع الجهاز - Device Type Filtering

## الميزة الجديدة ✨

تم إضافة نظام فلترة متقدم لعرض الأجهزة حسب نوعها في صفحة الأجهزة.

## كيف تعمل الميزة

### 1. عرض أزرار الفلترة
في أعلى قائمة الأجهزة، يظهر:
- **زر "الكل"**: يعرض جميع الأجهزة
- **أزرار أنواع الأجهزة**: زر لكل نوع جهاز موجود في الصالة
- كل زر يعرض عدد الأجهزة من هذا النوع

### 2. الفلترة التفاعلية
- عند الضغط على نوع معين، تظهر فقط الأجهزة من هذا النوع
- الزر المختار يتلون بالبنفسجي (primary color)
- العنوان يتغير ليعرض النوع المختار وعدد الأجهزة
- الفلترة فورية بدون إعادة تحميل الصفحة

### 3. حالات خاصة
- إذا لم توجد أجهزة من النوع المختار، تظهر رسالة "لا توجد أجهزة من هذا النوع"
- إذا لم توجد أنواع أجهزة، لا تظهر أزرار الفلترة

## التصميم

### أزرار الفلترة:
```
┌─────────────────────────────────────────┐
│  🔲 نوع الجهاز                         │
│                                         │
│  [الكل (12)] [PS5 (5)] [PS4 (4)] ...  │
└─────────────────────────────────────────┘
```

### الألوان:
- **زر غير مختار**: outline style (حدود فقط)
- **زر مختار**: background بنفسجي + نص أبيض
- **Hover**: تأثير hover خفيف

## البنية التقنية

### Server Component (page.tsx)
- يجلب الأجهزة مع أنواعها من قاعدة البيانات
- يجلب الجلسات والحجوزات النشطة
- يستخرج أنواع الأجهزة الفريدة
- يمرر البيانات للـ Client Component

### Client Component (devices-grid-client.tsx)
- يدير حالة الفلتر المختار (useState)
- يفلتر الأجهزة حسب النوع المختار (useMemo)
- يعرض أزرار الفلترة
- يعرض الأجهزة المفلترة

## الأداء

### التحسينات:
- ✅ **useMemo** للفلترة - تجنب إعادة الحساب غير الضرورية
- ✅ **Client-side filtering** - فلترة فورية بدون طلبات للسيرفر
- ✅ **Server-side data fetching** - جلب البيانات مرة واحدة
- ✅ **Optimized queries** - جلب device types مع الأجهزة في استعلام واحد

## الاستخدام

### للموظفين وأصحاب الصالات:
1. افتح صفحة الأجهزة
2. شاهد أزرار الفلترة في الأعلى
3. اضغط على نوع الجهاز المطلوب
4. ستظهر فقط الأجهزة من هذا النوع
5. اضغط "الكل" للعودة لعرض جميع الأجهزة

## مثال على البيانات

### Device Types:
```typescript
[
  { id: "1", name: "PS5", name_ar: "بلايستيشن 5", name_en: "PlayStation 5" },
  { id: "2", name: "PS4", name_ar: "بلايستيشن 4", name_en: "PlayStation 4" },
  { id: "3", name: "PC", name_ar: "كمبيوتر", name_en: "PC Gaming" }
]
```

### Devices:
```typescript
[
  { id: "d1", name: "PS5-01", device_type_id: "1", status: "available" },
  { id: "d2", name: "PS5-02", device_type_id: "1", status: "active" },
  { id: "d3", name: "PS4-01", device_type_id: "2", status: "available" }
]
```

## الدعم متعدد اللغات

- ✅ العربية: يستخدم `name_ar`
- ✅ الإنجليزية: يستخدم `name_en`
- ✅ التبديل التلقائي حسب لغة المستخدم

## الحالات الخاصة

### 1. لا توجد أجهزة:
```
┌─────────────────────────────┐
│   🖥️                        │
│   لا توجد أجهزة بعد        │
└─────────────────────────────┘
```

### 2. لا توجد أجهزة من النوع المختار:
```
┌─────────────────────────────┐
│   🖥️                        │
│   لا توجد أجهزة من هذا النوع│
└─────────────────────────────┘
```

### 3. نوع واحد فقط:
- تظهر أزرار الفلترة حتى لو كان نوع واحد
- يمكن التبديل بين "الكل" والنوع الوحيد
