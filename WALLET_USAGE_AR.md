# نظام المحافظ - دليل الاستخدام

## نظرة عامة
نظام المحافظ يسمح لأصحاب الصالات والموظفين بإنشاء وإدارة محافظ للاعبين (المسجلين والضيوف) لكل صالة على حدة.

## كيفية استخدام المحافظ

### 1. الوصول لصفحة المحافظ
- من القائمة الجانبية، اضغط على "المحافظ"
- ستجد صفحة البحث عن المحافظ وإدارتها

### 2. إنشاء أو البحث عن محفظة

**ملاحظة مهمة**: المحفظة تُنشأ تلقائياً عند أول عملية شحن! لا تحتاج لإنشائها يدوياً.

#### للاعبين الضيوف (Guest Players):
1. اختر "Guest Player" من الأزرار في الأعلى
2. أدخل اسم الضيف **بالضبط** كما يظهر في الحجوزات
   - مثال: إذا كان الحجز باسم "أحمد"، اكتب "أحمد"
3. اضغط "Search"
4. إذا لم تكن المحفظة موجودة:
   - سيظهر "⚠️ New wallet - will be created on first top-up"
   - أدخل المبلغ واضغط "Create Wallet & Add $XX.XX"
   - سيتم إنشاء المحفظة وإضافة المبلغ في عملية واحدة

#### للاعبين المسجلين (Registered Players):
1. اختر "Registered Player" من الأزرار
2. أدخل Username الخاص باللاعب (مثل: ahmed123)
   - يمكنك الحصول عليه من صفحة المستخدمين في لوحة الإدارة
3. اضغط "Search"
4. إذا لم تكن المحفظة موجودة:
   - سيظهر "⚠️ New wallet - will be created on first top-up"
   - أدخل المبلغ واضغط "Create Wallet & Add $XX.XX"
   - سيتم إنشاء المحفظة وإضافة المبلغ في عملية واحدة

### 3. شحن المحفظة (Top Up)

بعد العثور على المحفظة أو إنشائها:

1. اضغط على زر "Top Up Wallet"
2. أدخل المبلغ المراد إضافته (بالدولار)
3. أضف ملاحظة اختيارية (مثل: "باقة شهرية"، "هدية عيد ميلاد")
4. اضغط "Add $XX.XX" للتأكيد
5. سيتم إضافة المبلغ فوراً وتحديث الرصيد

### 4. استخدام المحفظة في الدفع

عند إنهاء جلسة لاعب لديه محفظة:

1. افتح نافذة الجلسة (Session Modal) بالضغط مرتين على الجهاز النشط
2. ستظهر رصيد المحفظة في الأعلى باللون الأخضر
3. أدخل سعر الساعة العادي (Cash Rate)
4. اختيارياً: أدخل سعر الساعة للمحفظة (Wallet Rate) - إذا تركته فارغاً سيستخدم السعر العادي
5. اختر طريقة الدفع:
   - **Full Cash**: كل شيء نقداً (الجلسة + المنتجات)
   - **Wallet + Cash**: الجلسة من المحفظة، المنتجات نقداً
6. اضغط "Confirm Payment"

**ملاحظة مهمة**: المنتجات دائماً تُدفع نقداً، المحفظة تغطي تكلفة الجلسة فقط!

## أنواع اللاعبين

### اللاعبون المسجلون (Registered Players)
- لديهم حساب في التطبيق
- يتم التعرف عليهم بـ Username (فريد لكل لاعب)
- يمكنهم الحجز من التطبيق
- المحفظة مرتبطة بحسابهم

### اللاعبون الضيوف (Guest Players)
- زبائن عاديون بدون حساب
- يتم إنشاء حجز لهم من قبل الموظف
- يتم التعرف عليهم بالاسم فقط
- المحفظة مرتبطة بالاسم

## معلومات مهمة

### الرصيد
- الرصيد يُحسب من مجموع كل العمليات (ليس رقم مخزن)
- كل عملية شحن أو خصم تُسجل في قاعدة البيانات
- هذا يضمن إمكانية تتبع كل عملية

### نطاق المحفظة
- كل محفظة خاصة بصالة واحدة
- نفس اللاعب يمكن أن يكون له محافظ مختلفة في صالات مختلفة
- الرصيد لا ينتقل بين الصالات

### الأسعار
- **Cash Price**: السعر العادي للساعة (نقداً)
- **Wallet Price**: سعر اختياري مخفض عند الدفع بالمحفظة
- إذا لم تحدد Wallet Price، سيستخدم النظام Cash Price

### الأمان
- فقط أصحاب الصالات والموظفين يمكنهم شحن المحافظ
- كل عملية تُسجل مع اسم الموظف الذي قام بها
- اللاعبون يمكنهم فقط رؤية محافظهم (لا يمكنهم الشحن)

## أمثلة عملية

### مثال 1: شحن محفظة لاعب مسجل
```
1. اختر "Registered Player"
2. اكتب "ahmed123" (Username)
3. اضغط Search
4. اضغط "Top Up Wallet"
5. أدخل المبلغ: 50
6. الملاحظة: "باقة أسبوعية"
7. اضغط Add
✅ تم إضافة 50$ لمحفظة ahmed123
```

### مثال 2: شحن محفظة ضيف
```
1. اختر "Guest Player"
2. اكتب "محمد"
3. اضغط Search
4. اضغط "Top Up Wallet"
5. أدخل المبلغ: 100
6. الملاحظة: "باقة 20 ساعة"
7. اضغط Add
✅ تم إضافة 100$ لمحفظة محمد
```

### مثال 2: دفع جلسة بالمحفظة
```
1. محمد يلعب على جهاز PC-01
2. اضغط مرتين على الجهاز لفتح نافذة الجلسة
3. رصيد المحفظة: 100$
4. السعر العادي: 5$ / ساعة
5. سعر المحفظة: 4$ / ساعة (خصم)
6. مدة الجلسة: 2 ساعة
7. أضاف مشروب بـ 2$
8. اختر "Wallet + Cash"
9. النتيجة:
   - تكلفة الجلسة: 8$ (من المحفظة)
   - المشروب: 2$ (نقداً)
   - الرصيد المتبقي: 92$
   - المبلغ المطلوب نقداً: 2$
```

### مثال 3: رصيد غير كافٍ
```
1. رصيد اللاعب: 5$
2. تكلفة الجلسة: 10$
3. عند اختيار "Wallet + Cash" سيظهر:
   ❌ "Insufficient wallet balance"
4. الحل: إما شحن المحفظة أو اختيار "Full Cash"
```

## الأسئلة الشائعة

**س: هل يمكن استخدام المحفظة لشراء المنتجات؟**
ج: لا، المحفظة تغطي تكلفة الجلسة فقط. المنتجات دائماً نقداً.

**س: ماذا لو كتبت اسم الضيف بشكل مختلف؟**
ج: سيتم إنشاء محفظة جديدة! تأكد من كتابة الاسم بالضبط كما في الحجوزات.

**س: هل يمكن نقل الرصيد بين اللاعبين؟**
ج: لا، حالياً كل محفظة مستقلة.

**س: هل يمكن استرجاع المبلغ من المحفظة؟**
ج: نعم، يمكن إضافة ميزة الاسترجاع (Refund) لاحقاً.

**س: كيف أعرف Username للاعب مسجل؟**
ج: من لوحة الإدارة → Users → انظر لعمود Username في جدول المستخدمين.
